# Technical Specification

## Роли

### Диспетчер
- Видит транспорт в реальном времени
- Отслеживает задержки

### Аналитик
- Анализирует историю
- Строит графики

## User Stories

1. Как диспетчер, я хочу видеть транспорт на карте, чтобы контролировать движение.

2. Как аналитик, я хочу спросить:
"Построй график загрузки маршрута №555"
→ получить визуализацию.

3. Как пользователь, я хочу написать:
"Покажи опаздывающие автобусы"
→ увидеть их на карте.
