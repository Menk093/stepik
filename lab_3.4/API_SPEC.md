# API Specification и архитектура приложения

## Выбранное API

Используется API новостей NewsAPI.

---

## Основные параметры API

**Base URL:**
https://newsapi.org/v2

**API-ключ:**
Требуется. Передаётся как query-параметр:
apiKey=YOUR_API_KEY

**Основной эндпоинт:**
/top-headlines

**Параметры запроса:**

* country (например: ru)
* category (например: technology)
* apiKey (обязательный)

---

## Пример успешного ответа

```json
{
  "status": "ok",
  "totalResults": 100,
  "articles": [
    {
      "title": "Название новости",
      "description": "Описание",
      "urlToImage": "image.jpg",
```

---

## Ограничения API

* Ограничение на количество запросов (rate limit)
* Бесплатный тариф имеет лимиты

# Проектирование приложения

## Цель приложения

Показывать список актуальных новостей по категории "Технологии".

---

## Структура компонентов

* App — основной компонент
* NewsCard — карточка новости
* Loader — индикатор загрузки
* ErrorAlert — отображение ошибки

---

## Состояние (State)

* articles: [] — список новостей
* isLoading: boolean — состояние загрузки
* error: string | null — ошибка

---

## Логика работы

1. Пользователь открывает страницу
2. Показывается Loader
3. Выполняется запрос к API
4. Если успешно:

   * отображаются новости
5. Если ошибка:

   * отображается ErrorAlert
