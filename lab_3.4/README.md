Создай модуль newsService.js для работы с News API (v2).

Используй fetch.

Функция fetchTopNews должна:

* делать GET-запрос на https://newsapi.org/v2/top-headlines
* использовать параметры country, category, apiKey
* проверять response.ok
* выбрасывать ошибку при неудаче
* возвращать JSON при успехе


Почему нельзя хардкодить API-ключ:

👉 Ответ:

ключ будет доступен всем пользователям
его могут украсть и использовать
возможна блокировка API
переменные окружения обеспечивают безопасность


👉 Что происходит при отключении интернета:

При отключении сети fetch не может выполнить запрос.
Возникает ошибка, которая перехватывается в try...catch.

В результате:

устанавливается состояние error
отображается ErrorAlert
пользователь видит сообщение и кнопку "Повторить"

👉 Достаточно ли этого:

Частично.
Можно улучшить:

добавить понятный текст ошибки
показать подсказку про интернет
