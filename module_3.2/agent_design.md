🔹 План выполнения (Agent Plan)
Получить доступ к репозиторию через GitHub API
Найти и прочитать package.json
Проверить зависимости через API (например npm audit / Snyk)
Сформировать отчёт (список уязвимостей)
Отправить отчёт в Slack через Webhook
🔹 Используемые инструменты
GitHub API
npm audit API / Snyk API
Slack Webhook API
🔹 Возможные ошибки
❌ Ошибка авторизации (GitHub / Slack)
❌ API недоступен
❌ Нет зависимостей
❌ Неверный формат ответа
❌ Ошибка отправки сообщения
🔹 Как агент должен это учитывать
retry (повтор запроса)
обработка ошибок (try/catch)
fallback (например: "нет уязвимостей")
логирование
